Título de la actividad: El Escáner de Hidrocarburos: Detectando Enlaces

Edad objetivo: 15-16 años (2° Medio)

Nivel de dificultad: Intermedio (Nivel 2)

Objetivos educacionales:

  • Química: Reconocer las diferencias estructurales entre alcanos, alquenos y alquinos basándose en la saturación de los enlaces de carbono.
  • Pensamiento Computacional: Utilizar lógica de eventos y condicionales múltiples para categorizar información basada en la entrada de sensores.

Contexto: Eres un científico en una expedición espacial que ha encontrado muestras de compuestos orgánicos desconocidos. Tu labor es construir un "Escáner de Hibridación" que identifique si un hidrocarburo es saturado (alcano) o insaturado (alqueno/alquino) basándose en una clave de inclinación. Los estudiantes construirán modelos moleculares físicos (usando esferas de plumavit o palitos) y colocarán el dispositivo sobre ellos. Según cómo orienten el sensor (representando el ángulo de enlace de la geometría molecular: tetraédrica 109.5°, trigonal plana 120° o lineal 180°), el sistema identificará el tipo de hidrocarburo.

Dispositivos a utilizar:

  • Input: Sensor de Inclinación (Inclination), Botón Táctil (Touch Button).
  • Output: Matriz LED (LED Matrix), Reproductor de Sonido (Audio Player), Lámpara (Lamp).

Solución: Instrucciones para implementar la actividad

  1. Construcción: El alumno debe tener tres modelos de cartón o plástico que representen un alcano (enlace simple), un alqueno (enlace doble) y un alquino (enlace triple).
  2. Operación:
    • Al presionar el Botón Táctil, el escáner se activa.
    • Si el dispositivo está en posición Horizontal (Inclinación 0°), el sistema detecta un Alcano (Enlace simple).
    • Si el dispositivo se inclina a la Izquierda, detecta un Alqueno (Enlace doble).
    • Si se inclina a la Derecha, detecta un Alquino (Enlace triple).
  3. Feedback: La Matriz LED mostrará el número de enlaces (1, 2 o 3 líneas) y el Reproductor de Sonido dirá el nombre del grupo funcional.

Pseudocódigo de la solución:

START
  // Inicialización del sistema
  Draw on [LED Matrix 1] with 7x7 pattern (empty square)
  Set color of [Lamp 1] to hex value #ffffff

  MAIN LOOP:
    // El escaneo solo ocurre si se mantiene el botón presionado
    IF get property 'pressed' from [Touch Button 1] THEN
      Let angulo = get property 'value' from [Inclination 1]
      
      // Lógica de identificación de hidrocarburos
      IF angulo == 'flat' THEN
        // ALCANO - Enlace Simple
        Draw on [LED Matrix 1] with 7x7 pattern (one horizontal line)
        Execute play on [Audio Player 1] with sound 'alkane_detected.mp3'
        Set color of [Lamp 1] to hex value #00ff00 // Verde para saturado
      
      ELSE IF angulo == 'left' THEN
        // ALQUENO - Enlace Doble
        Draw on [LED Matrix 1] with 7x7 pattern (two horizontal lines)
        Execute play on [Audio Player 1] with sound 'alkene_detected.mp3'
        Set color of [Lamp 1] to hex value #ffff00 // Amarillo para insaturado
        
      ELSE IF angulo == 'right' THEN
        // ALQUINO - Enlace Triple
        Draw on [LED Matrix 1] with 7x7 pattern (three horizontal lines)
        Execute play on [Audio Player 1] with sound 'alkyne_detected.mp3'
        Set color of [Lamp 1] to hex value #ff0000 // Rojo para muy insaturado
      END IF
      
    ELSE
      // Estado de espera
      Draw on [LED Matrix 1] with 7x7 pattern (dot in center)
      Execute stop on [Audio Player 1]
    END IF

    Wait 150 milliseconds
  END MAIN LOOP
END

  2. Programar una variable conteo que aumente con cada toque.
  3. Establecer una ventana de tiempo (ej. 500ms) para decidir qué galaxia es según los toques recibidos.

Pseudocódigo de la solución:

START
  // Inicializamos la variable de conteo
  Let toques = 0

  MAIN LOOP:
    // Esperamos a que alguien toque la textura
    ON EVENT click of [Touch Button 1]:
      Let toques = toques + 1
      
      // Esperamos un breve momento para ver si hay más toques
      Wait 500 milliseconds
      
      IF toques == 1:
        // Caso: Galaxia Elíptica (Estrellas viejas/rojas)
        Set color of [Lamp 1] to hex value #FF4500

        Set color of [Lamp 1] to hex value #00FF00
